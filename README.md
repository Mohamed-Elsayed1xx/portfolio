# Mohamed Elsayed — Portfolio

A personal portfolio website built to showcase my full-stack projects, technical skills, and professional profile. Designed with a dark space-themed UI, interactive 3D elements, and smooth animations.

**Live:** [portfolio-flax-three-18.vercel.app](https://portfolio-flax-three-18.vercel.app)

---

## Features

- Interactive 3D robot powered by Spline (reacts to mouse movement)
- Animated typing effect cycling through roles
- Orbiting planet with real photo avatar in the About section
- Animated rocket that flies between planets
- Project cards with live demo and GitHub links
- Skills grid with color-coded tech badges
- Contact section with WhatsApp, LinkedIn, and GitHub links
- Fully responsive across all screen sizes
- Browser translation protection (no React DOM conflicts)
- Star background canvas animation

## Getting Started

```bash
# 1. Clone the repo
git clone https://github.com/Mohamed-Elsayed1xx/portfolio.git
cd portfolio

# 2. Install dependencies
npm install

# 3. Start the dev server
npm run dev

# 4. Build for production
npm run build
```
